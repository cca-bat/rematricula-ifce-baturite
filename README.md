# Rematrícula — IFCE Campus Baturité

Portal público de apoio à renovação de matrícula dos estudantes veteranos do IFCE Campus Baturité.

O site reúne:

- tutorial completo de Pedido, Ajuste e Ajuste Extraordinário de Matrícula;
- calendário das etapas;
- disciplinas e horários ofertados em 2026.2 e, para Letras S7 a S9, em 2026.1;
- salas, professores, pré-requisitos, equivalências e observações importantes;
- orientações sobre o reajuste excepcional via SEI.

## Atualização

Os dados do portal são gerados a partir das bases acadêmicas conferidas pela CCA. A planilha do mapa de salas, guia **Base para lançamento**, é a referência principal para ofertas e horários.

Para atualizar o conteúdo:

1. substitua ou revise as bases de origem;
2. execute o gerador dos dados do portal;
3. confira as alterações no site e no tutorial;
4. gere novamente a versão estática;
5. publique os arquivos resultantes na raiz da branch `main`.

O endereço do GitHub Pages permanece o mesmo depois das atualizações.

## Privacidade

Não são publicados nomes, matrículas nem planilhas de acompanhamento individual. Alertas derivados dessas bases são apresentados apenas de forma geral e orientativa.
